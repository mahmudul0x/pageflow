# PageFlow Pro

A modern Facebook Page management tool for creators, marketers, and agencies.

## Features

- **Multi-page posting** — Publish to all your Facebook Pages at once
- **Smart scheduling** — Plan your content calendar with an intuitive scheduler
- **Deep analytics** — Track reach, engagement, and growth across every page

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Zustand (state management)
- TanStack React Query
- React Router v6
- Recharts (analytics charts)
- Axios (HTTP client)

## Getting Started

```bash
npm install
npm run dev
```

The app runs in mock mode by default (no backend required). To connect a real backend, set the `VITE_API_BASE_URL` environment variable to your API base URL.

## Scripts
